# Customer-Churn-Prediction


This project predicts which customers are likely to **churn (leave a company)** using ML models such as Logistic Regression, Random Forest, and XGBoost.

It includes:
- Data preprocessing
- Feature engineering
- EDA (Exploratory Data Analysis)
- Model training and tuning
- Model evaluation
- API deployment using Flask
